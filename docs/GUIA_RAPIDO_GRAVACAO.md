## 📹 PASSO 2: Instalar e Configurar Loom (5 min)

### 2.1 Instalar
```
1. Abrir: https://loom.com
2. Criar conta grátis (usar seu e-mail)
3. Baixar: Loom Desktop App (recomendado)
   Ou: Extensão Chrome
```

### 2.2 Testar Microfone
```
1. Abrir Loom
2. Clicar no ícone de microfone
3. Falar "teste 1, 2, 3"
4. Ouvir playback
5. Ajustar volume se necessário
```

### 2.3 Configurações
```
Clicar em Settings (engrenagem):
  ✓ Quality: HD 1080p
  ✓ Microphone: [Seu microfone]
  ✓ Camera: Optional (pode desligar se quiser)
  ✓ Record System Audio: OFF
```

## 🎥 PASSO 4: GRAVAR! (3-5 min de gravação)

### 4.1 Iniciar Gravação Loom

```

2. Clicar "New Video"
3. Escolher:

   - Ou "Screen + Camera" (tela + sua webcam)
4. Selecionar: Janela do navegador (localhost:3000)
5. Clicar "Start Recording"
6. Countdown: 3... 2... 1... 
```

### 4.2 Seguir Script

**TER O SCRIPT VISÍVEL!**

```
Abrir: SCRIPT_VIDEO_SIMPLES.txt

Ler linha por linha enquanto:
- Navega pelo sistema
- Clica nos botões
- Mostra as telas

NÃO PRECISA DECORAR!
Pode ler naturalmente.
```

### 4.3 Sequência de Telas (o que clicar)


## 🆘 Troubleshooting

### "Loom não grava tela"
```
Solução:
- Windows: Dar permissão de gravação de tela
- Configurações → Privacidade → Gravação de tela
- Habilitar Loom
```

### "Áudio muito baixo"
```
Solução:
- Loom Settings → Microphone
- Aumentar volume de entrada
- Ou falar mais perto do microfone
```

### "Errei muito, desisti"
```
Solução:
- NORMAL! Todo mundo erra
- Não precisa ser perfeito
- Respirar e continuar
- Ou gravar em partes menores (30s cada)
```

### "Não sei o que falar"
```
Solução:
- Literalmente LER o script
- Não precisa decorar
- Pode parecer que está lendo, TUDO BEM
- Autenticidade > Performance de ator
```
