## 🛠️ COMO GRAVAR NO LOOM

### 1. Instalar Loom
```
1. Acesse: https://loom.com
2. Criar conta grátis
3. Instalar extensão Chrome
4. Ou baixar app desktop (recomendado)
```

### 2. Preparar Gravação

**Antes de começar:**
- [ ] Sistema rodando em localhost:3000
- [ ] Dados de teste prontos:
  - [ ] 1 produto cadastrado
  - [ ] 3 fornecedores cadastrados
  - [ ] 1 requisição aprovada (para demo)
  - [ ] 1 cotação com propostas (para comparar)
- [ ] Fechar abas desnecessárias
- [ ] Aumentar zoom do navegador (125-150%)
- [ ] Modo claro (light mode) - melhor visibilidade
- [ ] Ensaiar 2-3 vezes (não precisa ser perfeito)

### 3. Configurações Loom

```
Tipo de gravação: "Screen + Cam" ou só "Screen"
Qualidade: HD 1080p
Microfone: Testar antes (ambiente silencioso)
Webcam: Opcional (recomendo SIM - gera conexão)
```

### 4. Gravar

**Dicas:**
- ✅ Falar devagar e com clareza
- ✅ Pausar 1 segundo entre ações (facilita edição)
- ✅ Se errar, continuar (dá pra cortar depois)
- ✅ Sorrir na webcam (transmite confiança)
- ✅ Olhar para câmera no início e fim

**Erros comuns:**
- ❌ Falar rápido demais
- ❌ Clicar sem explicar
- ❌ Muito silêncio (dead air)
- ❌ Gírias ou "ahm", "éh"

### 5. Editar no Loom

Após gravar, Loom permite:
- ✂️ Cortar partes (início/fim)
- ⏩ Acelerar trechos (ex: login)
- 🎵 Adicionar música de fundo (opcional)
- 📝 Adicionar CTAs clicáveis no vídeo
- 👆 Emojis e anotações

**Edições obrigatórias:**
1. Cortar os primeiros 2-3 segundos (você abrindo navegador)
2. Acelerar login (2x speed)
3. Adicionar CTA final: "Teste grátis" (link clicável)

## 💡 DICAS EXTRAS

### Para Parecer Profissional:
1. **Iluminação:** Grave durante o dia perto de janela
2. **Fundo:** Parede lisa ou desfocada (não bagunça)
3. **Roupa:** Casual profissional (não precisa terno)
4. **Energia:** Sorrir e falar com entusiasmo (mas não exagerar)

### Para Economizar Tempo:
1. Gravar de manhã (mente mais fresca)
2. Não buscar perfeição (autenticidade > produção Hollywood)
3. Se errar, continuar (corta depois)
4. Fazer 2-3 takes completos e escolher o melhor

### Para Aumentar Conversão:
1. Mostrar resultado antes de explicar (AIDA: Attention primeiro)
2. Usar números específicos (20%, 10h/semana)
3. Comparar "antes" vs "depois" (planilhas vs sistema)
4. Terminar com CTA claro ("Teste grátis" não "Saiba mais")

## 📊 MÉTRICAS PARA ACOMPANHAR

Loom mostra analytics:

- **View Rate:** Quantos % assistem até o fim
  - Meta: >40% assistem tudo
  
- **Drop-off Points:** Onde param de assistir
  - Otimizar essas partes se <50% continua

- **Clicks nos CTAs:** Quantos clicam "Teste grátis"
  - Meta: >5% de click-through

**Se views são baixas mas conversão é alta:** Vídeo bom!
**Se views são altas mas conversão é baixa:** CTA fraco ou público errado
